# Blog Archive Manager

過去のブログで書き溜めた記事（エクスポートログ）をブラウザ上で読み込み、快適に統合・検索・閲覧できるオープンソースのビューワーツールです。

## 🚀 Live Demo

### 直接ブラウザで動作を試すことができます：

(https://siro-pj-tech.github.io/blog-archive/)

## ✨ 主な機能

- 複数ブログの統合: 異なるブログサービスのログファイルを順次読み込み、1つのアーカイブに統合可能。

- スマートフィルタ: キーワード検索にヒットした年号だけを自動抽出してドロップダウンに表示。

- JSONエクスポート: 統合したデータを1つのJSONファイルとして保存でき、次回から一瞬で復元可能。

- 回想コピー: 過去記事の引用に便利な「タイトル・日付・本文」の整形テキストコピー機能。

- 完全スタンドアロン: サーバー不要。``index.html`` 1枚でどこでも動作します。

## 📖 使い方

1. 公開URLへアクセスするか、``index.html`` をダウンロードしてブラウザで開きます。

2. 画面右上の 「ログを追加」 ボタンをクリック。

3. ブログからエクスポートしたログファイル（``.log`` 等）を読み込むか、テキストを貼り付けます。

4. 左側のパネルから「年号」や「キーワード」で記事を探して閲覧します。

## 🛠 対応フォーマット

Movable Type (MT) 形式をベースとしたログに対応しています。
```
TITLE: 記事タイトル
DATE: 04/20/2026 12:00:00
-----
BODY:
記事の本文
-----
--------

```
